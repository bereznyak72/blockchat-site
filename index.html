Особенности BlockChat

Уникальные возможности нашего P2P мессенджера

Важная информация

  • Работает в локальной сети (LAN) НОВОЕ: вне локальной сети через Radmin VPN
  • Требует стабильного IP для подключения
  • Сигнальный сервер автоматически запускается на порту 8080
  • Поддержка только текстовых сообщений (пока!)

Безопасность и приватность

Ваши сообщения никогда не хранятся на серверах. Вся переписка остается только между участниками беседы.

Прямое P2P соединение

После установки связи через сигнальный сервер, сообщения передаются напрямую между устройствами без промежуточных серверов.

Мгновенная доставка

Сообщения доставляются мгновенно благодаря прямому соединению между пользователями. Никаких задержек на серверах.

Поддержка Markdown в сообщениях

Форматируйте текст легко и быстро! BlockChat поддерживает Markdown-разметку для создания жирного и курсивного текста, списков, цитат и многого другого.

Простой интерфейс

Интуитивно понятный интерфейс позволяет легко настроить соединение и начать общение без сложных настроек.

Кроссплатформенность

Доступно для Windows, Linux и macOS. Общайтесь с пользователями разных операционных систем.

Как подключиться

Простой процесс установки защищенного P2P-соединения

1

Запустите BlockChat на обоих устройствах

Сигнальный сервер запускается автоматически при необходимости. Больше не нужно запускать его вручную!

2

Обменяйтесь IP-адресами

Один пользователь становится инициатором (подключается), другой - ресивером (ожидает подключения). Обменяйтесь IP-адресами.

3

Установите P2P соединение

После обмена сигналами через автоматически запущенный сервер устанавливается прямое соединение между устройствами.

4

Начните общение

Наслаждайтесь безопасным общением напрямую! Сообщения передаются без промежуточных серверов.

Советы по подключению:

  • Для связи вне локальной сети используйте Radmin VPN или аналогичные решения
  • Убедитесь, что брандмауэр разрешает соединения на порту 8080
  • Используйте статические IP-адреса для стабильного соединения
