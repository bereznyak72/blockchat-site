Принципы конфиденциальности BlockChat

BlockChat — это децентрализованный P2P мессенджер, созданный с принципом полной приватности пользователей. В отличие от традиционных мессенджеров, мы не храним и не имеем доступа к вашим личным данным и сообщениям.

BlockChat не хранит ваши сообщения на серверах. Все сообщения передаются напрямую между устройствами пользователей через зашифрованные P2P соединения.

Какие данные мы НЕ собираем

В отличие от большинства мессенджеров, BlockChat сознательно не собирает:

  • Персональные данные пользователей (имя, email, номер телефона)
  • Контакты из вашей адресной книги
  • Историю сообщений и переписки
  • Метаданные сообщений (время отправки, получатели)
  • Геолокационные данные
  • Информацию об устройствах в сети
  • IP-адреса пользователей (кроме временной обработки для установления P2P соединения)
  • Файлы, передаваемые между пользователями

Ваши сообщения и файлы никогда не проходят через наши серверы и не хранятся в каком-либо централизованном хранилище. Они существуют только на устройствах отправителя и получателя.

Техническая реализация приватности

BlockChat использует следующие технологии для обеспечения приватности:

P2P Архитектура

Приложение устанавливает прямое соединение между устройствами пользователей без промежуточных серверов для передачи данных.

Сигнальный сервер

Временный сигнальный сервер используется только для установления P2P соединения между пользователями и не имеет доступа к содержимому сообщений или файлов.

Групповые чаты

В групповых чатах сообщения передаются через децентрализованную сеть участников. Информация о участниках хранится только во время активной сессии чата.

Шифрование и безопасность

BlockChat использует современные методы шифрования для защиты ваших данных:

Сквозное шифрование (End-to-End Encryption)

Все сообщения и файлы шифруются на устройстве отправителя и расшифровываются только на устройстве получателя с использованием алгоритма AES-GCM с ключом 256 бит.

Обмен ключами шифрования

Ключи шифрования генерируются случайным образом для каждой сессии и передаются через установленное P2P соединение.

Шифрование файлов

Все передаваемые файлы автоматически шифруются перед отправкой и расшифровываются только на устройстве получателя.

Хранение данных

BlockChat обеспечивает локальное хранение всех данных:

  • История сообщений хранится только на вашем устройстве
  • Полученные файлы сохраняются только с вашего явного согласия
  • Настройки приложения хранятся локально на вашем устройстве
  • При переустановке приложения все данные безвозвратно удаляются

Сторонние услуги и интеграции

BlockChat не использует сторонние аналитические сервисы, рекламные сети или трекеры, которые могли бы собирать ваши данные.

Единственное исключение — службы распространения приложений (такие как GitHub Releases), которые могут собирать анонимную статистику загрузок в соответствии со своими политиками конфиденциальности.

Обновления и поддержка

Приложение проверяет наличие обновлений через официальный сайт BlockChat. При проверке передается только номер версии вашего приложения.

Сообщения об ошибках и запросы функций отправляются добровольно через GitHub Issues и содержат только информацию, которую вы явно указали в отчете.

Изменения в политике конфиденциальности

Мы можем периодически обновлять эту политику конфиденциальности. О любых изменениях мы будем уведомлять через:

  • Официальный сайт BlockChat
  • Релиз-ноты новых версий приложения
  • Уведомления в самом приложении

Продолжение использования BlockChat после внесения изменений означает ваше согласие с обновленной политикой конфиденциальности.
